注意事项

1 最好先获取数据,再渲染swiper.
用vue往往都是先声明一个空数组占位, 异步获取数据后,再添加到数组中.
但对于swiper, 最好用v-if包裹, 等到获取完数据后在渲染swiper.

如果把最外层的v-if去掉,用update更新swiper,可能会有bug.
