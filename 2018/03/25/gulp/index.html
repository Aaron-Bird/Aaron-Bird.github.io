API

src 监听文件位置

gulp.src(globs[, options])

  • globs 文件路径

常用的options

options.buffer

是否返回buffer,默认为true
为false时返回stream, 这在处理大文件时很有用
注: 部分插件可能不支持stream

options.read

默认值为true.为false不读取文件,返回null

options.base

设置输出文件的路径结构

dest 输出文件

输出文件,可通过pipe多次调用
gulp.dest(path[, options])

task 创建任务

gulp.task([name,] fn)

  • name 任务名
  • fn 被执行的函数,或gulp.series gulp.parallel gulp.lastRun

调用copy任务

series 依序运行多个任务

gulp.series(...tasks)
依序调用一系列任务/函数.当发生错误时,后边的任务不会被执行

parallel 并行运行多个任务

gulp.parallel(...tasks)
类似series,区别是parallel并行调用任务.即使有一个任务出错,后边的函数都会被执行
如果一堆任务中有watch,建议使用parallel
因为用series的话,watch会一直运行下去,导致后面的任务无法被执行

watch 监控文件

gulp.watch(globs[, opts][, fn])
监控文件,当发生变化时,调用fn
