屏蔽智联招聘上的虚假招聘

2017-12-17

智联招聘上总是有皮包公司发一堆虚假招聘,可以用uBlock Origin扩展屏蔽掉
规则 sou.zhaopin.com##.newlist:has-text(公司名称)

// 例如想屏蔽 四十大盗有限公司 北极熊有限公司
// 就把下列规则填在uBlock origin的自定义规则列表里,一行一个
sou.zhaopin.com##.newlist:has-text(四十大盗有限公司) 
sou.zhaopin.com##.newlist:has-text(北极熊有限公司)
